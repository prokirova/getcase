# GetCase

Платформа для участия студентов в реальных кейсах от компаний.

Студенты могут:
- просматривать актуальные кейсы
- принимать участие в понравившихся задачах
- отправлять свои решения (файлы)

Компании (организаторы) публикуют кейсы с описанием, дедлайнами и требуемыми навыками.

**Текущий статус проекта:** MVP (минимально жизнеспособный продукт)

## Основные возможности

- Регистрация и авторизация студентов
- Просмотр списка всех кейсов
- Детальная страница кейса (описание, компания, направления, дедлайн)
- Принятие участия в кейсе
- Отправка решения (загрузка файла до 50 МБ)
- Страница «Мои кейсы» (список задач, в которых участвуешь)
- Профиль студента с редактированием данных и навыков

## Технологии

**Backend**
- Python 3.10+
- Flask
- MySQL / MariaDB
- mysql-connector-python
- bcrypt (хэширование паролей)
- сессии (flask.session)

**Frontend**
- HTML + CSS (градиентный дизайн, адаптивность минимальная)
- Vanilla JavaScript (без фреймворков)
- Fetch API для общения с бэкендом

**База данных**
- MySQL
- Таблицы: Students, Companies, Cases
