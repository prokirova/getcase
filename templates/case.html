Начать работу
Разработать мобильное приложение для инвестиций с функциями регистрации, просмотра портфеля, покупки/продажи акций и пополнения счёта. Спроектировать UI/UX-дизайн всех экранов в Figma с учётом мобильных паттернов и доступности. Реализовать клиентскую часть на Flutter/React Native с интеграцией API брокера для получения котировок в реальном времени. Настроить безопасную авторизацию с двухфакторной аутентификацией и биометрией, подключить платёжный шлюз для транзакций. Провести тестирование функционала, исправить баги и подготовить демо-версию для iOS и Android. Оформить исходный код на GitHub, написать техническую документацию и защитить проект перед заказчиком с презентацией результатов. Разработать систему уведомлений о сделках и изменениях рынка. Реализовать образовательный модуль со статьями и тестами для начинающих инвесторов. Обеспечить соответствие требованиям 152-ФЗ и стандартам безопасности финансовых приложений. Подготовить презентацию проекта с демонстрацией ключевого функционала и метрик производительности.
